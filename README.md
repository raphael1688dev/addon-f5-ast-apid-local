# F5 AST API Discovery (Local)

**Installation:**
Add this URL to Home Assistant Add-on Store:
`https://github.com/raphael1688dev/addon-f5-ast-apid-local`

**Setup:**
1. Install and start **General ClickHouse**.
2. Install **F5 AST Logic**.
3. In Configuration, set `db_host` to your NUC's IP address (e.g., 192.168.1.X) and `db_port` to `19000`.
